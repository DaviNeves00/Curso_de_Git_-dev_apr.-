Git init: primeiro código a ser digitado

Git status: para ver o status do Git e quais arquivos foram adicionados 
equais não foram.

Git add + "nome do arquivo": para adicionar um arquivo no Git

Git add .: para adicionar todos arquivos não adicionados

Git push: para subir os arquivos para nuvem

Git remote add origin + "lind do repositório do GitHub": para selecionar o repositório que vo~ê deseja subir
o arquivo
 
Git relog: para analisar as alterações/commits feitos
